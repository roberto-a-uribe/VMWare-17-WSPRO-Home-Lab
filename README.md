# 🛠️ Install and Configure VMWare 17 Workstation PRO for Home Lab


## 💡Project Showcase: Virtual Infrastructure Deployment
**Objective:** This showcase demonstrates the architecture and implementation of a localized virtualization environment using **VMware Workstation 17 Pro**. The project highlights the deployment of a versatile multi-OS ecosystem, ranging from enterprise-grade directory services to specialized security and open-source platforms.

---

## 🛡️[Install and Configure VMWare for Home Lab ](https://github.com/roberto-a-uribe/VMWare-17-WSPRO-Home-Lab/tree/main/VMWare-17-Install)


### 🔐 Environment Architecture
The setup serves as a high-fidelity sandbox designed to simulate real-world networking and administrative scenarios. Key components include:

* **Infrastructure Core:** **Windows Server** configured with **Active Directory Domain Services (AD DS)** to manage centralized authentication and group policies.
* **Client Endpoints:** Integrated **Windows 11** instances for testing domain integration and workstation management.
* **Linux Ecosystem:** * **Ubuntu:** Serving as a robust platform for general-purpose development and server-side workflows.
    * **Kali Linux:** Utilized as a dedicated environment for penetration testing and network security auditing.

### Technical Scope
This project covers the end-to-end lifecycle of virtual machine management, including:
* Hypervisor configuration and resource optimization.
* Virtual networking (NAT, Bridged, and Host-Only) for secure inter-VM communication.
* Cross-platform interoperability within a unified hardware environment.
